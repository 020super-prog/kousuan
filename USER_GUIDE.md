# 📖 小学口算助手 - 使用指南

## 欢迎使用小学口算助手！

这是一款专为小学生设计的口算练习工具，帮助孩子提升计算能力，让学习变得更有趣！

点击首页的练习按钮，快速开始练习：

1. **➕ 加法练习** - 适合1-2年级
2. **➖ 减法练习** - 适合1-2年级  
3. **✖️ 乘法练习** - 适合2-3年级

首页底部显示：
- 📝 **今日练习**: 今天完成的题目数量
- 📊 **正确率**: 今天的答题正确率
- 📚 **待复习错题**: 需要重新练习的错题数量

## ✏️ 口算练习 - 答题过程

### 开始答题

1. **看题目**: 屏幕中央显示大号题目，例如 "36 + 47 = ?"
2. **输入答案**: 使用下方数字键盘输入答案
3. **提交**: 点击绿色的"✓"按钮提交答案

### 数字键盘
```
[ 1 ]  [ 2 ]  [ 3 ]
[ 4 ]  [ 5 ]  [ 6 ]
[ 7 ]  [ 8 ]  [ 9 ]
[ ← ]  [ 0 ]  [ ✓ ]
```
- **数字键**: 输入数字
- **← 删除键**: 删除最后一位数字
- **✓ 确认键**: 提交答案

### 答题反馈

**答对了** ✅
- 出现绿色对勾动画
- 手机会震动（如果开启了震动）
- 自动进入下一题

**答错了** ❌
- 出现红色叉号动画
- 显示正确答案
- 错题自动保存到错题本
- 1秒后自动进入下一题

### 顶部信息
- **左上角**: 当前进度，如 "5/20"（第5题，共20题）
- **右上角**: 用时，如 "02:35"（2分35秒）

## 📊 练习结果 - 查看成绩

### 成绩展示

完成练习后，会显示：

**大圆圈分数**

1. **查看错题** - 查看本次答错的题目（如果有错题）

## 📚 错题本 - 管理错题

### 查看错题

错题本自动收集所有答错的题目，帮助你针对性复习。

### 筛选功能

顶部可以按题型筛选：
- **全部** - 显示所有错题
- **加法** - 只显示加法错题
- **减法** - 只显示减法错题
- **乘法** - 只显示乘法错题
- **除法** - 只显示除法错题
- **混合** - 只显示混合运算错题

### 错题卡片

每个错题显示：
- **题目**: 原始题目
- **你的答案**: 你当时的错误答案（红色删除线）
- **正确答案**: 标准答案（绿色加粗）
- **日期**: 出错时间
- **题型标签**: 题目类型

### 操作按钮

**单题操作**
- **🔄 重练**: 重新做这道题（开发中）
- **✅ 已掌握**: 标记为已掌握，从错题本移除

**批量操作**
1. 勾选"全选"或单独勾选错题
2. 点击"生成练习卷" - 针对选中的错题生成新的练习（开发中）
3. 点击"删除" - 批量删除选中的错题

## 📸 拍照批改 - 智能批改（开发中）

### 使用步骤
1. 点击底部"拍照批改"Tab
2. 将纸质作业放在取景框内
3. 点击"拍照批改"或"从相册选择"
4. 等待识别（3-5秒）
5. 查看批改结果

### 注意事项
- 确保光线充足
- 作业字迹清晰
- 完整拍摄所有题目

## 💡 使用技巧

### 提高正确率
1. **认真审题** - 看清楚是加法还是减法
2. **不要着急** - 想清楚再输入答案
3. **多练习** - 熟能生巧，多练几次就会提高
4. **复习错题** - 定期查看错题本，避免重复犯错

### 高效练习
1. **设定目标** - 每天练习20题
2. **记录进步** - 关注正确率变化
3. **循序渐进** - 从简单题型开始，逐步提高难度
4. **坚持不懈** - 每天练习，养成好习惯

### 家长辅导建议
1. **鼓励为主** - 多表扬孩子的进步
2. **适度练习** - 每次15-20分钟即可
3. **查看统计** - 关注孩子的学习数据
4. **针对性复习** - 利用错题本重点练习

## ❓ 常见问题

### Q1: 题目太简单/太难怎么办？
**A**: 使用"自定义"功能（开发中），可以设置难度、数值范围等参数。

### Q2: 错题本的题目可以删除吗？
**A**: 可以。点击"已掌握"按钮，或使用批量删除功能。

### Q3: 数据会丢失吗？
**A**: 不会。数据保存在本地，除非卸载小程序或清除缓存。

### Q4: 可以离线使用吗？
**A**: 可以！题目生成和答题功能完全本地化，无需网络。只有登录、拍照批改等功能需要网络。

### Q5: 如何关闭音效和震动？
**A**: 进入"我的" → 功能设置 → 关闭对应开关。

### Q6: 分享功能在哪里？
**A**: 完成练习后，在结果页面底部有"分享成绩"按钮。

## 🎯 学习目标建议

### 一年级（加减法）
- **每天**: 练习20题
- **目标**: 正确率80%以上
- **范围**: 20以内加减法

### 二年级（加减乘）
- **每天**: 练习30题
- **目标**: 正确率85%以上
- **范围**: 100以内加减法，9以内乘法

- **每天**: 练习40题
- **目标**: 正确率85%以上
- **范围**: 1000以内加减法，乘除法表

- **每天**: 练习50题
- **目标**: 正确率90%以上
- **范围**: 多位数运算，混合运算
